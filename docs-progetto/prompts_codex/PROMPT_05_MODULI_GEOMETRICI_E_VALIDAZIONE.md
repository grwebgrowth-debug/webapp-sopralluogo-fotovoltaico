# Prompt 05 — Moduli geometrici e validazione

Obiettivo:
Creare i moduli puri di geometria e validazione, senza mescolarli ai componenti UI.

Devi fare:
- generazione falda rettangolare
- generazione falda trapezoidale
- generazione falda triangolare
- generazione quadrilatero guidato
- ostacolo rettangolare e circolare
- applicazione margine di sicurezza
- funzioni di validazione “dentro / fuori falda”

File da leggere:
- 05_REGOLE_GEOMETRICHE.md

Criterio chiave:
La logica del triangolo deve rispettare:
- distanza dall’angolo destro della base
- altezza dalla base (H)
