# Prompt 01 — Scaffold progetto e architettura

Obiettivo:
Creare il progetto base Next.js + TypeScript + Tailwind con una struttura cartelle pulita e coerente con i documenti di progetto.

Puoi fare:
- scaffold app
- struttura cartelle
- layout base
- file placeholder
- configurazione minima

Non fare ancora:
- logica geometrica
- integrazione n8n
- integrazione Google Sheet
- wizard completo
- form definitivi

File da leggere:
- 01_SPECIFICA_V1_BLOCCATA.md
- 03_SPECIFICA_FUNZIONALE.md

Output atteso:
- progetto avviabile
- cartelle `components`, `features`, `lib`, `types`
- naming coerente e in italiano lato UI
