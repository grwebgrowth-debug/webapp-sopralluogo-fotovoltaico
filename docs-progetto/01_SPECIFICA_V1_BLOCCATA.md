# Specifica V1 bloccata

## Dentro la V1
- Web app wizard multi-step
- Dati cliente e sopralluogo
- Selezione tipo di tetto
- Compilazione di più falde
- Forme falda supportate:
  - rettangolare
  - trapezoidale
  - triangolare
  - quadrilatero irregolare guidato
- Inserimento ostacoli:
  - rettangolari
  - circolari
- Tipi ostacolo:
  - camino
  - lucernario
  - sfiato
  - antenna / palo
  - area non utilizzabile
  - altro ostacolo
- Selezione pannello da Google Sheet
- Revisione tecnica completa prima dell’invio
- Invio finale a n8n
- Misure in centimetri

## Fuori scope V1
- CAD libero
- Drag&drop avanzato
- Simulazione energetica
- Multiutente avanzato
- Storage documentale interno
- Database dedicato
- AI come motore di posizionamento geometrico

## Step definitivi
1. Dati cliente e sopralluogo
2. Tipo di tetto
3. Falde
4. Ostacoli
5. Pannello
6. Revisione tecnica
7. Invio a n8n
